## Features
- Opens current track directly inside `Rdio.app` when clicking the title;
- Sound bars animation pauses when Rdio is paused.

## Commands
```rb
# Playstate
Play
Pause
Toggle

# Track
Next
Previous

# Collection
Add
Remove

# Mobile
Sync
Unsync
```
